作成日: 2026-06-04

## 概要

タイムド箇条書きメモツール。1分間という制約の中でテーマに対する考えを書き出す思考整理ツール。  
Python + Flask でローカルサーバーを立て、バッチ起動でブラウザが開く。

## アーキテクチャ

### バックエンド（app.py）

| エンドポイント | メソッド | 処理 |
|--------------|--------|------|
| `/` | GET | index.html を返す |
| `/api/count` | GET | 今日の枚数を返す `{"count": 4}` |
| `/api/count/increment` | POST | カウンター +1 して `{"count": 5}` を返す |

- `localhost:5000` のみでリッスン（外部公開しない）
- `data/counter.json` に `{"date": "YYYY-MM-DD", "count": N}` を保存
- 日付が変わっていたら count を 0 にリセット

### フロントエンド（index.html）

単一 HTML ファイル。CSS・JS はインライン。外部 CDN 不使用（オフライン動作）。

## 機能仕様

### タイトル入力

- 1行テキスト入力
- アンダーライン型スタイル（border-bottom のみ）

### メモエリア（箇条書き）

| キー操作 | 動作 |
|---------|------|
| フォーカス時 | 先頭に「・」が入った状態で開始 |
| `Enter` | 次の行に「・」を自動挿入 |
| `Shift + Enter` | 全角スペース1個でインデントして改行（「・」に位置を揃える） |
| `Backspace` | 「・」または全角スペースだけの行で前の行に戻る |

### タイマー

- メモエリアへの入力開始で自動スタート（1分 = 60秒）
- プログレスバーで残り時間を視覚化
- **残り15秒**: タイマーエリアが赤くなり点滅アニメーション
- **0秒**: メモエリアを `readonly` に変更 → ダイアログ表示

### ダイアログ（1分経過）

- メッセージ:「1分経過しました」
- ボタン: OK（クリックでダイアログを閉じる）
- OK後もメモエリアは非活性のまま

### コピーボタン

- メモエリアの内容をクリップボードにコピー（`navigator.clipboard.writeText`）
- コピー成功時: ボタンラベルを一時的に「✅ コピー済」に変更

### クリアボタン

- タイトル・メモをクリア
- タイマーをリセット（00:60に戻す）
- メモエリアを再び活性化・「・」を初期挿入
- **タイマーが 0 に達した後のクリアのみ** `/api/count/increment` を POST してカウンターを +1
  （タイマー未完了のままクリアした場合はカウントしない）

### サイドバー統計

| 項目 | 内容 |
|------|------|
| TIME | 残り秒数 + プログレスバー |
| LINES | 現在の行数（リアルタイム更新） |
| TODAY | 今日の完了枚数（サーバーから取得） |

## 起動フロー（start.bat）

1. `python app.py` をバックグラウンド起動
2. 1〜2秒待機（サーバー起動を待つ）
3. `start http://localhost:5000` でブラウザを開く

## セキュリティ

- localhost のみ（`app.run(host='127.0.0.1')`）
- 保存データは日付と数値のみ（個人情報なし）
- API キー・認証情報なし
- `.gitignore` 対象: `data/`, `__pycache__/`, `*.pyc`
